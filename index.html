About Code Wars


Welcome to Code Wars! This is the new website for the codewars. You can find more information and previous iterations of the code wars in the tabs. The challenges can be found inside the Challenges tab.


In the Challenges page, you will find 8 challenges to complete. To view each challenge you must click on the card of the challenge and you will be redirected to the challenge page. The instruction of each individual challenge can be found inside the challenge's pages The challenges must be solved in sequence (i.e you need to solve first challenge to get link to second challenge and so on). The teams that solves all the challenges first will be the winner! Good Luck!!
